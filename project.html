Linda's ICT Project

ICT project introduction

Project name:Creation of personal website

Values and Methods

Values：

●Responsible

●Integrity and honesty

●Creative and competitive

●Adhere to customer-oriented

Methods：

1. Open a website host (provide sufficient website space, server, and ensure the necessary hardware...for example: CPU, Hard drive, Memory and Monitor).

2. Install the website building system (website source code, website program...for example: Adobe Dreamweaver CC, Codepen and Notepad++).

3. Improve the content of the website (add the navigation keys on the home page of the website, the pictures in the website, the font, size, color and position of the text...).

4. Save the code and export (compress the code and save it to the specified location...).

5. Test the website (make sure the code is correct).

6. Register the website domain name (confirm the website address and URL...).

Required hardware and software

Hardware Details
1.CPU At least 2G.
2.Memory The memory with ECC check must be more than 1G.
Software Details
1.Adobe Dreamweaver CC The memory with ECC check must be more than 1G.
2.Notepad++ Notepad++ is lightweight and we can use the mobile version to write code on the go without having to install it into Windows.
